# leinad-sch.github.io

My Github Pages profile site.

85% vibe-coded.

Features:

* 100% static, no backend, no framework
* blog posts are in Markdown, rendered using `marked.js`
* `giscus`comments

Pages:

* **Home**
* **Blog:** a very simple blog
* **Projects page**
* **CV page:** my résumé
